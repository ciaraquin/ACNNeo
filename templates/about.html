About

This project is a secure social media platform built with Flask and Python. Users can create accounts, post messages, and interact with other users in a secure environment.

On account creation, users will be given an RSA key pair for encryption and decryption.

Groups will be given unique AES keys for secure communication. Each group member will have access to their group's AES key by decrypting it with their own private key.
